XResetScreenSaver

Syntax

XResetScreenSaver(display)
      Display *display;

Arguments

display Specifies the connection to the X server.

Description

The XResetScreenSaver() function resets the screen saver.

See also

XSetScreenSaver(), XForceScreenSaver(), XActivateScreenSaver(), XGetScreenSaver(), "Screen Saver Control".
