XFreeFontPath

Syntax

XFreeFontPath(list)
      char **list;

Arguments

list Specifies the array of strings you want to free.

Description

The XFreeFontPath() function frees the data allocated by XGetFontPath().

See also

XSetFontPath(), XGetFontPath(), XListFonts(), XLoadFont(), "Setting and Retrieving the Font Search Path".
