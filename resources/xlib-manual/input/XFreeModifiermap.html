XFreeModifiermap

Syntax

XFreeModifiermap(modmap)
        XModifierKeymap *modmap;

Arguments

modmap Specifies the XModifierKeymap structure.

Description

The XFreeModifiermap() function frees the specified XModifierKeymap structure.
