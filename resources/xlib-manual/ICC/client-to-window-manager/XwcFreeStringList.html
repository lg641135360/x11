XwcFreeStringList

Syntax

void XwcFreeStringList(list)
      wchar_t **list;

Arguments

list Specifies the list of strings to be freed.

Description

The XwcFreeStringList() function frees memory allocated by XwcTextPropertyToTextList().

XmbTextListToTextProperty(), XwcTextListToTextProperty(), XmbTextPropertyToTextList(), XwcTextPropertyToTextList(), XDefaultString(), XSetTextProperty(), XStringListToTextProperty(), "Converting String Lists".
