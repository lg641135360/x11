XVisualIDFromVisual

Syntax

VisualID XVisualIDFromVisual(visual)
       Visual *visual;

Arguments

visual Specifies the visual type.

Description

The XVisualIDFromVisual() function returns the visual ID for the specified visual type.

See also

XGetVisualInfo(), XMatchVisualInfo(), "Visual Types"
