XShrinkRegion

Syntax

XShrinkRegion(r, dx, dy)
      Region r;
      int dx, dy;

Arguments

r Specifies the region.
dx
dy 		Specify the x and y coordinates, which define the amount you want to shrink the specified region.

Description

Positive values shrink the size of the region, and negative values expand the region.

See also

XCreateRegion(), XDrawRectangle(), XEmptyRegion(), XIntersectRegion(), XOffsetRegion(), XShrinkRegion(), XSubtractRegion(), XUnionRectWithRegion(), XUnionRegion(), XXorRegion(), "Moving or Shrinking Regions".
