XDestroyRegion

Syntax

Diagnostics

XDestroyRegion(r)
      Region r;

Arguments

r Specifies the region.

Description

XDestroyRegion() destroys the region.

See also

XCreateRegion(), XEmptyRegion(), XIntersectRegion(), XSetRegion(), "Creating, Copying, or Destroying Regions".
