XEmptyRegion

Syntax

Bool XEmptyRegion(r)
      Region r;

Arguments

r Specifies the region.

Description

The XEmptyRegion() function returns True if the region is empty.

See also

XCreateRegion(), XEqualRegion(), XIntersectRegion(), XPointInRegion(), XRectInRegion(), "Determining if Regions Are Empty or Equal".
