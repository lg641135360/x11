XEqualRegion

Syntax

Bool XEqualRegion(r1, r2)
      Region r1, r2;

Arguments

r1
r2 		Specify the two regions.

Description

The XEqualRegion() function returns True if the two regions have the same offset, size, and shape.

See also

XCreateRegion(), XEmptyRegion(), XEqualRegion(), XIntersectRegion(), XPointInRegion(), XRectInRegion(), "Determining if Regions Are Empty or Equal".
