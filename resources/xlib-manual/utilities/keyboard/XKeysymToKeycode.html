XKeysymToKeycode

Syntax

KeyCode XKeysymToKeycode(display, keysym)
      Display *display;
      KeySym keysym;

Arguments

display Specifies the connection to the X server.
keysym Specifies the KeySym that is to be searched for.

Description

If the specified KeySym is not defined for any KeyCode, XKeysymToKeycode() returns zero.

See also

XConvertCase(), XKeycodeToKeysym(), XKeysymToString(), XLookupKeysym(), XStringToKeysym(), Keyboard Utility Functions.
